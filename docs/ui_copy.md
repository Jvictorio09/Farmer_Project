# AgriTrack UI Copy Guide

## Confirmations
- **Delete**: “Delete this record? This can’t be undone.”
- **Edit / Save**: “Save changes to this record?”
- **Logout**: “Log out now?”

## Reminders
- Success: “Reminder saved.” / “Reminder updated.” / “Reminder deleted.”
- Error: “Please provide both a reminder message and due date.”

## Expenses
- Success (add/edit/delete): “Expense recorded.” / “Expense updated successfully.” / “Expense deleted.”
- Filters helper: “Showing: {Month or ‘All months’} • {Year if set}”
- Chart helper: “Tap a point to see exact amount.”

## Activities
- Success (add/edit/delete): “Activity logged successfully. Forecast generated.” / “Activity updated successfully.” / “Activity deleted.”
- Planting helper: “We’ll compute min/est/max yield and harvest window after you save.”

## Auth
- Logout success: “You have been logged out.”
- Password reset request: “If that email exists, we’ve sent a link to reset your password.”
- Forgot password helper: “Pop in the email linked to your AgriTrack account. We’ll send a reset link right away.”
- Password reset status: “Reset link sent” / “Set a new password” / “Password updated”

## Chart Labels
- Line chart: “Monthly Expenses — {Year}”; X-axis “Month”; Y-axis “Amount (₱)”
- Donut chart: “Expenses by Category”
- Harvest rail: “Upcoming Harvests”

## Tone Notes
- Friendly, clear, and concise (“Tagalog-English mix ok: e.g., ‘Tap a point to see exact amount.’”)
- Prefer actionable headlines first, explanation afterward.
- Technician banner: “Technician mode · farmer dashboards stay unchanged”
